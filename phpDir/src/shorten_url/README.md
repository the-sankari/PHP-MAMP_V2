# URL Shortener

This project is a simple URL shortener that allows users to input a long URL and receive a shortened version of it. The shortened URL can then be used to access the original site.

## Features

- Input field for long URL
- Button to shorten URL
- Display of shortened URL
- Button to go to shortened URL
- Display of date and time when shortened URL was created
- Toast notification to display messages to the user

## Technologies Used

- PHP
- MySQL
- PHPMyAdmin
- HTML
- CSS
- Bootstrap

## Setup

To use this project, follow these steps:

1. Clone the repository to your local machine.
2. Set up a local development environment with PHP and MySQL.
3. Create a database and import the provided SQL file.
4. Update the database connection details in the `config.php` file.
5. Run the project in your web browser.

## Usage

To use the URL shortener, follow these steps:

1. Enter a long URL in the input field.
2. Click the "Shorten" button.
3. The shortened URL will be displayed on the page.
4. Click the "Go to site" button to visit the original site using the shortened URL.
5. The date and time when the shortened URL was created will be displayed below the URL.
